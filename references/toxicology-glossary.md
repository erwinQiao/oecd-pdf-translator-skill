# OECD Toxicology and Experimental Terminology Glossary

This glossary provides accurate English-Chinese translations for OECD test guideline documents, with equal priority for all terminology domains.

## Toxicology Terms (毒理学术语)

| English | Chinese | Notes |
|---------|---------|-------|
| phototoxicity | 光毒性 | Toxic response from chemical + light exposure |
| cytotoxicity | 细胞毒性 | Toxicity to cells |
| photo-cytotoxicity | 光细胞毒性 | Phototoxicity affecting cells |
| IC50 / IC 50 | IC50（半抑制浓度） | Half maximal inhibitory concentration |
| Photo-Irritation-Factor | 光刺激因子 | PIF, measures phototoxic potential |
| PIF | PIF（光刺激因子） | Abbreviation for Photo-Irritation-Factor |
| Mean Photo Effect | 平均光效应 | MPE, alternative phototoxicity measure |
| MPE | MPE（平均光效应） | Abbreviation for Mean Photo Effect |
| Neutral Red Uptake | 中性红摄取 | NRU assay for cell viability |
| NRU | NRU（中性红摄取） | Abbreviation for Neutral Red Uptake |
| Neutral Red | 中性红 | Vital dye used in viability assays |
| NR | 中性红 | Abbreviation for Neutral Red |
| irradiance | 辐照度 | Light intensity (W/m² or mW/cm²) |
| irradiation | 辐照 | Exposure to radiation/light |
| dose of light | 光剂量 | Quantity of light (J/cm²) |
| molar extinction coefficient | 摩尔消光系数 | MEC, measures light absorption |
| MEC | MEC（摩尔消光系数） | Abbreviation for molar extinction coefficient |
| photoreactive | 光反应性的 | Reacts to light exposure |
| phototoxicant | 光毒性物质 | Chemical causing phototoxicity |
| photogenotoxicity | 光遗传毒性 | DNA damage from chemical + light |
| photoallergy | 光过敏 | Allergic reaction to chemical + light |
| photocarcinogenicity | 光致癌性 | Cancer-causing potential with light |
| lysosome | 溶酶体 | Cell organelle containing enzymes |
| Reactive Oxygen Species | 活性氧 | ROS, cause oxidative damage |
| ROS | ROS（活性氧） | Abbreviation for Reactive Oxygen Species |
| xenobiotic | 异生物 | Foreign chemical substance |
| viability | 活力 | Ability of cells to survive/function |
| cell viability | 细胞活力 | Measure of live/healthy cells |
